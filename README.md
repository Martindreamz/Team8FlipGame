# Team8FlipGame
Team 8 card flipping memory game

This is an Android game application.

## Application Structure
### Activities:
  - menu
  - imagePicking
  - game
  - leaderboard
  
### Service 
  - DBService for score
  - BGMusicService

### Models
  - image
  - score
  
### Adaptors
  - imageAdaptor
  - scoreAdaptor
